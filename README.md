# README  
## 開発言語  
- Ruby3.0.1  
- Rails6.1.6  
## 就職termの技術  
- devise  
- Ajaxを使ったコメント機能  
- お気に入り機能  
## カリキュラム外の技術  
- ransack  
## 実行手順  
$git clone git@github.com:Jakky-enru/MultiMom.git  
$cd MultiMom  
$bundle install  
$yarn install  
$rails db create && rails db:migrate  
$rails s
